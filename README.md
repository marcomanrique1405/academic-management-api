# Academic Management Api
API REST desarrollada con Spring Boot para administrar un sistema académico.
Permite gestionar alumnos, docentes, tutores, materias, grupos e inscripciones.

El proyecto está diseñado para demostrar buenas prácticas de desarrollo backend,
incluyendo arquitectura en capas, principios SOLID, manejo centralizado de
excepciones, persistencia en base de datos H2 en memoria y documentación clara
con Swagger/OpenAPI.
---
## Objetivo
- Gestion de Alumnos.
- Manejo de excepciones.
----
## Tecnologías utilizadas
- Java 17
- Spring Boot 3.2.5
- Spring Web
- Spring Data JPA
- Spring Validation
- H2 Database 
- Swagger / OpenAPI 
- Maven
---
### Requisitos
- Java 17 
- Maven 3.8.7
